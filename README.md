# Veleda-GitOps-Based-Microservices-Platform-on-Kubernetes
Veleda: GitOps-Based Microservices Platform  Designed and deployed a cloud-native microservices platform using Kubernetes, Docker, GitLab CI/CD, and ArgoCD. Implemented GitOps-based continuous deployment, automated container image management, Kubernetes networking, ingress routing, and service-to-service communication.
